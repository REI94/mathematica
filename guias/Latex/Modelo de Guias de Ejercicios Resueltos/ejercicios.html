  1. Demostrar que (A ∩ B) ∪ C = (A ∪ C) ∩ (B ∪ C) (Propiedad Distributiva)
    Para demostrar esta igualdad debemos probar primero que,
    Por simbología tenemos,
    Por logíca tenemos que p → q, donde p es: x ∈ (A ∩ B) ∪ C y q sera: x ∈ (A ∪ C) ∩ (B ∪ C), de modo que es necesario demostrar esta proposición.
    Entonces, por definición de la unión entre dos conjuntos se tiene que,

    x ∈ (A ∩ B) ó x ∈ C

    Mostraremos que,

    Ahora bien,

    1) Tenemos que x ∈ C, entonces x ∈ (A ∪ C) y x ∈ (B ∪ C). Por unión x ∈ A ó x ∈ C y x ∈ B ó x ∈ C ambas son verdaderas porqué estamos suponiendo que x ∈ C, por lo tanto x ∈ (A ∪ C) ∩ (B ∪ C)
    2) Tenemos x ∈ (A ∩ B), entonces x ∈ A y x ∈ B. Tomando estas afirmaciones se tiene que x ∈ (A ∪ B) por unión x ∈ A ó x ∈ C por las afirmaciones esto es cierto y también tenemos x ∈ (A∪C) ⇒ x ∈ B ó x ∈ C. Por las afirmaciones esto también es cierto porqué x ∈ B, por lo tanto x ∈ (A ∪ C) ∩ (B ∪ C) es verdadera.
    Ahora falta demostrar,

    Procedemos igual,
    Tenemos que x ∈ (A ∪ C) y x ∈ (B ∪ C), ahora consideremos 2 casos x ∈ C ó xC
    Mostremos que,
    1) Supongamos que x ∈ C. Entonces x ∈ (A ∩ B) ∪ C es verdadera
    2) Supongamos que xC. Entonces x ∈ (A∪C) necesariamente se tiene que x ∈ A. De igual manera, ya que x ∈ (B ∪ C), entoces x ∈ B con esto mostramos que x ∈ (A ∩ B) y por lo tanto x ∈ (A ∩ B) ∪ C.
  2. Demostrar que A ⊂ B ↔ B′⊂ A′
    Para demoestrar esto debemos probar que,

    A ⊂ B → B′⊂ A′ y B′⊂ A′→ A ⊂ B

    Tenemos que, por lógica una proposición condicional es lógicamente igual a su contrarecíproca, quiere decir p → q ⇔¬q →¬p

    Por esto tenemos que,
    1) Si B′⁄⊂ A′→ A ⁄⊂ B
    Ahora procedemos a demostrar 1), supongamos que A y B son conjuntos tales que B′⊂ A′ y mostremos que A ⁄⊂ B, como por hipótesis B′⁄⊂ A′. Entonces, existe un elemento x que pertenece a B′ pero no a A′. Es decir, existe un x ∈ B′ y xA′. En otras palabras, existe x tal que xB y x ∈ A, esto precisamente dice que A ⁄⊂ B y así damos por demoestrado B′⁄⊂ A′→ A ⁄⊂ B.
    2)

    B′⊂ A′→ A ⊂ B, aplicando lo anterior tenemos
    A ⁄⊂ B → B′⁄⊂ A′

    Tenemos como hipótesis que existe un x ∈ A pero xB, esto queire decir que xA′ y x ∈ B′, esto es precisamente que B′⁄⊂ A′, por la hipótesis que x ∈ B′, hemos demostrado A ⁄⊂ B → B′⁄⊂ A′.
